## 剪辑手法

- **稳定人像底盘**：主持人不是频繁换景，而是作为可信讲解者贯穿全片。
- **证据画面插入**：平台截图、价格页、地图、日历用于证明“不是空讲概念”。
- **短 B-roll 替换**：景区人流、户外家庭场景用于承接假日、旺季、出游等关键词。
- **常驻主题标题**：左上大标题多次出现，形成系列感。
- **底部字幕同步**：几乎全程有字幕，便于静音观看。
- **关键词卡片化**：把复杂建议拆成“套餐”“当地特色”等可执行词。
- **镜头节奏**：大约 3-6 秒出现一次视觉变化，但人物口播保持连续。

## 对系统的启发

建议新增或强化一个成片方式：`竖版口播增强模板`。

这个模板不是纯数字人全程一张脸，也不是纯图文视频，而是：

1. 数字人/真人口播占 60%-70%；
2. B-roll 和截图证据占 20%-30%；
3. 关键词卡片、清单、编号占 10%-20%；
4. 全程保留统一标题区、字幕区和品牌身份区；
5. AI 自动根据脚本分镜决定什么时候插入截图、场景、卡片。

## 可落地到系统的字段

- `topic_title`：常驻标题，例如“旺季心法 / 一招制胜”
- `speaker_scene`：人物口播场景，如酒店办公室、客房、大堂、洽谈区
- `evidence_assets`：平台截图、订单截图、价格页、地图、日历
- `b_roll_keywords`：客流、前台、入住、客房、商圈、景区、团队会议
- `keyword_cards`：套餐、当地特色、提前预售、连住优惠、私域复购
- `subtitle_style`：底部白字、轻描边、关键字金色/蓝色
- `cut_frequency_seconds`：3-6 秒视觉变化一次

## 适合我们生成的酒店行业模板

用于“酒店无卡取电/智能客控/酒店加盟/旺季营销”时，可套用以下结构：

1. 开场：负责人出镜，一句话指出痛点或机会。
2. 证据：插入真实酒店场景、前台/房间/能耗/订单截图。
3. 方法：用三点清单讲方案。
4. 场景：用 B-roll 或 AI 实景演示客人入住、退房、客控联动。
5. 结果：回到负责人总结，强调适用条件和落地路径。
