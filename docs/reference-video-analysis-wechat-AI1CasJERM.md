## 一句话结论

这条视频不是纯 AI 实景视频，而是“真人/类真人口播 + 固定爆款标题栏 + 自动字幕 + AI/PPT 辅助解释页”的知识获客模板。我们要做到这个程度，核心不是先追求复杂实景，而是先把数字人口播、模板包装、字幕、插图页和分镜剪辑稳定打通。

- 主标题两行：第一行白字，第二行黄字，字体加粗并带阴影。
- 中部主画面：真人口播或解释页，约占画面高度 64%。
- 底部固定信息栏：深蓝底，约占画面高度 17%。
- 底部左侧有竖排“AI生成”标识。

- 中部字幕：白色大字，带黑/灰描边和阴影，通常一行 10-16 字。

## 脚本结构

1. 痛点钩子：金税四期、高企严查、台账不规范。
2. 标题承诺：夯实高企与加计扣除基础，吃透研发台账辅助账。
3. 方法拆解：一次性讲透，列出科创项目合规核心要点。
4. 证据材料：人员薪酬、设备折旧、试验耗材、设备使用记录、付款凭证。
5. 合规目标：账实、账证、账账相符。
6. 政策依据：委托研发、加计扣除口径、其他费用比例等。
7. 行动引导：评论区留言。

## 镜头和节奏

- 全片不是高频炫技剪辑，而是“口播为主，解释页打断”的稳定节奏。
- 每 5-10 秒有一次视觉变化，避免用户疲劳。
- 人物口播段占比约 60%，解释页/资料页占比约 40%。
- 人物始终保持同一身份、同一服装、同一外景背景，信任感稳定。
- 插图页通常保留一个小人物抠像，避免画面从“人讲”断成“纯PPT”。

## 技术实现判断

这类视频需要以下生产层：

1. 人物口播层
   - 可以是真人实拍，也可以是高质量数字人。
   - 必须有连续身体动作、自然手势和口型同步。
   - 单张头像驱动无法达到这个效果。

2. 模板包装层
   - 固定顶部标题栏。
   - 固定底部品牌/身份栏。
   - 自动生成大字幕。
   - 支持姓名、职位、卖点标签。

3. 辅助画面层
   - AI插图页、政策卡片、表格页、流程页。
   - 人物小抠像可以出现在左下角。
   - 辅助页只服务解释，不抢主线。

4. 剪辑编排层
   - 根据脚本自动分段。
   - 口播段和解释页交替出现。
   - 每段字幕跟随语音切换。

## 我们当前系统差距

- 目前“图文草稿预览”完全不能作为正式成片。
- Seedance 实景镜头可用，但更适合做 B-roll，不适合替代整条知识口播主线。
- 数字人口播必须接入真实数字人驱动，不能只用头像。
- 需要支持“口播模板成片”，而不是只支持“分镜实景成片”。
- 需要有一个参考视频解析结果，能反推模板参数。

## 系统要做到这个程度的能力清单

### 第一阶段：模板复刻

- 新增成片方式：`数字人口播模板`
- 模板参数：
  - 顶部双行标题
  - 标题颜色：白 + 黄
  - 底部身份条
  - 中部大字幕
  - 口播主画面比例
  - 插图页比例
  - 小人像角标
- 输出 60 秒竖版视频。

### 第二阶段：真实数字人

- 上传 1-3 分钟本人源视频。
- 支持声音克隆或指定 TTS 音色。
- 接入数字人驱动接口，生成连续口播。
- 如果没有源视频，不允许选择“真人数字人口播”。

### 第三阶段：AI 辅助页

- 根据脚本自动生成 3-5 张解释页。
- 支持政策卡、流程图、数据表、对比图、案例图。
- 可以调用 Seedance 或图片模型生成背景/B-roll。

### 第四阶段：自动剪辑

- 按脚本自动拆成：
  - 开场真人口播
  - 解释页
  - 真人口播
  - 资料页
  - 总结口播
- 自动挂字幕。
- 自动加顶部标题和底部身份。
- 自动生成封面。
