# A0AmHfFLLO 工程级逐帧解析

## 结论先行

这条参考视频不是数字人口播，也不是固定主持人视频。它是一条 **A/B 双人音频访谈可视化视频**：全片用酒店 B-roll 承载画面，用固定标题、实时波形、A/B 对话字幕和底部栏目品牌形成统一节目包装。

完整逐秒扫描没有发现可替换的何依依主持人层。画面里出现的人都是保洁、前台、住客、会议参与者等 B-roll 演员。因此：

- 可以工程化复刻它的结构、节奏、字幕、波形和栏目包装。
- 不应调用动作模仿、人像融合或单图数字人驱动来处理这条参考。
- 把黄丽放进画面不是“替换原人物”，而是新增主持人层，会改变参考片构图。
- 若坚持视觉结构一致，黄丽最合理的角色是 A 或 B 的授权声音，而不是画面主持人。

机器可读结果见 [a0amhfllo-exact-v1.json](./a0amhfllo-exact-v1.json)。

## 内容流程

全片不是“观点口播 + 随机素材”，而是主持人提问、嘉宾回答的三轮论证：

1. `0–28s`：抛出争议，24 小时退房是福利还是陷阱。
2. `28–98s`：解释排班、人工、周转效率和不同酒店类型的差异。
3. `98–144s`：给出有限试点、会员权益、阶梯收费和智能调度方案。
4. `144–234s`：进入第二主题，量化人工、清洁、能耗、房态和系统对接压力。
5. `234–279s`：进入第三主题，用调查数据判断是否为大众刚需。
6. `279–335s`：总结为细分需求，酒店应按自身条件选择。
7. `335–339s`：感谢收听、下期再见。

真实 ASR 按 115 秒切成三段完成，全文 `2087` 字。烧录字幕通过 macOS Vision 每 `0.5 秒` OCR 一次，原始得到 `360` 次文字变化，清洗后形成 `181` 条 A/B 字幕事件。OCR 时间精度为 `±0.5 秒`，正式复刻应以新配音的强制对齐时间戳为准。

## 音频判断

- 主体是两种可区分音色的连续对话，不是单人女声口播。
- 全片仅有 4 个约 `0.2–0.4 秒` 的短静音，说明对白密度很高。
- 未检测到可明确分离的背景音乐段，也未发现稳定的转场音效事件。
- 视觉节奏由对白轮次、真实振幅波形和硬切共同承担。
- 如果后续使用女声，建议黄丽作为 A 或 B 的其中一个声音，再配另一种明显可区分的自然音色。

## 为什么此前生成结果差

当前 `compose_wechat_reference_template()` 并没有复刻参考视频，它只是一个占位合成器：

| 当前实现 | 参考片实际做法 |
|---|---|
| 程序绘制一张模糊酒店房间背景 | 84 个真实酒店 B-roll 镜头 |
| 每 4 秒机械轮换 A/B 文本 | 字幕跟随真实说话轮次和语音时间戳 |
| 用正弦函数绘制假波形 | 波形来自真实混合音频振幅 |
| 额外生成程序化音乐 | 参考片没有明确可分离的配乐段 |
| 1280×720 上近似摆放图层 | 原片固定 1024×576 像素布局 |
| 单条 voiceover 拆成轮次 | 先生成结构化 A/B 对话脚本 |

所以问题不主要在视频模型提示词，而在合成架构走错了：系统用“单场景模板动画”冒充了“真实 B-roll 音频访谈剪辑”。再强的视频模型也无法替代错误的时间线和后期包装。

## 正确生产链路

系统应新增内部生产模式 `dialogue_broll_program`，不在页面增加额外复杂入口，由参考类型自动选择：

1. 主题生成结构化 A/B 对话，保留每轮 speaker、text、purpose。
2. A/B 分别使用两个自然音色合成，黄丽可绑定其中一个女声。
3. 将两轨语音合并，并取得句级或词级时间戳。
4. 依据每轮语义，从 owned/licensed 素材库检索酒店 B-roll。
5. 以 `3–5 秒` 为主分配镜头，在语义转折处硬切。
6. 后期按固定坐标渲染灰色标题、真实波形、A/B 字幕和自有栏目品牌。
7. 质检音画同步、字幕溢出、黑帧、素材版权、响度和镜头重复率。

这个链路不应调用人像融合、动作模仿或 Seedance 人物驱动。视频模型只在素材库缺少某个干净 B-roll 时补单个镜头，并且不得生成画面内文字。

## 可复刻边界

- **可做到结构级一致**：画幅、固定版式、镜头密度、硬切节奏、A/B 对话结构、波形、字幕颜色层级和栏目包装。
- **不能在不使用原素材时做到像素级一致**：原 B-roll、原音色、原文字和栏目品牌都是参考片资产。
- **发布版本必须原创替换**：使用自有酒店素材、自有 A/B 脚本、自有声音和自有栏目名称。
- **黄丽的正确接入方式**：绑定一个声音角色；若一定要出镜，应另做“主持人 + B-roll 插片”版本，不能称为这条参考的一比一复刻。

## 验收标准

- 画幅和固定图层误差不超过 `2px`（按 1024×576 基准）。
- 字幕切换与语音误差不超过 `120ms`。
- 波形必须来自最终混合音轨，不允许使用随机函数。
- 平均镜头长度控制在 `3.5–4.5 秒`，语义转折处切镜。
- 不连续重复同一素材超过 `8 秒`。
- 综合响度目标 `-12 ± 1 LUFS`，峰值不超过 `-1 dBTP`。
- 全片不得出现乱码、字幕溢出、异常黑帧或第三方水印。
