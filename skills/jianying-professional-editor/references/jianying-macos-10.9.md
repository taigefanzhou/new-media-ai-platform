# Mac 剪映专业版10.9执行规则

## 当前可靠链路

1. 打开或创建剪映原生草稿。
2. 通过 `Cmd+I` 导入本地素材。
3. 从“已添加素材”的缩略图中心长按约0.55秒。
4. 拖动到主时间线视频轨；蓝色框选不等于入轨。
5. 通过无障碍元素定位右上角导出按钮和导出确认按钮。
6. 监测剪映实际生成的 MOV/MP4，等待文件大小稳定后移动到目标路径。

执行脚本：

```bash
swift scripts/jianying_mac_automation.swift --doctor
swift scripts/jianying_mac_automation.swift \
  --media /absolute/path/input.mp4 \
  --output /absolute/path/output.mov \
  --replace
```

## 安全边界

- 运行前确认 macOS“辅助功能”已授权终端或执行宿主。
- 自动化过程中不要操作鼠标和键盘。
- 不直接修改 `draft_info.json` 或 `draft_meta_info.json`；剪映10.9草稿已加密。
- 不安装要求剪映5.9的自动导出器，不为兼容旧工具而降级生产剪映。
- 不删除或覆盖用户草稿；导出覆盖必须显式使用 `--replace`。
- 只有已授权或自有素材可以进入时间线。

## 当前能力边界

- 已验证：单素材导入、长按拖入主轨、打开导出、完成导出和文件校验。
- 多素材精剪应按剪辑决策表逐步扩展 UI 动作，并在每个动作后检查时间线状态。
- 美颜、智能抠图、自动字幕等 GPU/云端功能需单独定位当前版本 UI，不要声称已自动完成。
- 网页系统运行在服务器，本地剪映运行在用户 Mac；两者之间需要本机伴侣服务，不能由远程后端直接控制桌面。
