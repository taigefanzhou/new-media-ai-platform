# 专业短视频剪辑规则

## 1. 叙事与节奏

- 用一句话定义视频承诺。任何镜头若不推进承诺、证据、冲突或情绪，就删掉。
- 前3秒同时提供视觉主体、可读字幕和明确收益/悬念；重要选题生成至少两个钩子版本。
- 粗剪先按语义和情绪切点，不先加特效。删除停顿、口头禅、重复信息和无效反应。
- 镜头长度服从内容：高信息密度可快切，解释、表情和证据需要留出理解时间。
- 在动作发生中切镜头，或用相似构图、方向、颜色和声音完成匹配剪辑。

## 2. 镜头连续性

- J Cut：下一个场景声音先进入，再切换画面，用于预告人物、地点或信息。
- L Cut：上一场景声音延续到下一画面，用于维持连贯和隐藏生硬切点。
- 同一空间保持人物视线、运动方向和屏幕方向一致；改变方向时先给建立镜头。
- 连续切换时改变景别或角度，避免两个近似构图产生跳切；有意跳切必须服务于节奏。
- B-roll覆盖剪切点时，要与旁白当前名词、动作或证据对应，不能只追求“好看”。

## 3. 字幕与屏幕文字

- 按完整语义断句，每屏一到两行；短词不要孤立换行。
- 第一条字幕直接呈现承诺，不照抄“大家好，今天聊聊”。
- 普通字幕保持单一字体、稳定位置和高对比；关键词每屏最多强调一到两个。
- 技术词、人名、数字、单位和品牌名必须人工或规则复核。
- 竖屏字幕避开人物嘴部、底部平台按钮区和右侧互动区。

## 4. 音频

- 先清理和对齐人声，再加环境声、BGM和音效。
- 人声出现时对 BGM 做自动压低和短淡入淡出，避免突然抽空环境。
- 用环境声跨镜头维持空间连续，用短音效强调按钮、切换、警报、揭示和落点。
- 音效不得每个转场都出现；重复模板音会放大“AI成片感”。
- 导出前检查耳机和外放：对白清楚、无削波、左右声道正常、开头结尾无突变。

## 5. 调色与人像

- 顺序：曝光和白平衡校正 → 镜头间匹配 → 对比和饱和度 → HSL/曲线/色轮精修。
- 肤色优先保持真实；只做轻度均匀和瑕疵控制，不拉大眼、不瘦脸、不改变年龄感。
- 同一场景统一黑位、高光、色温和饱和度；滤镜强度宁低勿高。
- 图片和静态数字人可用轻微推拉或位移，但运动幅度要像真实摄影机。

## 6. 效果与转场

- 默认使用硬切、声音衔接和轻叠化。只有时间、空间、视角或情绪发生变化时才加显式转场。
- 关键帧优先用于信息引导：轻推近主体、跟随动作、突出界面区域、字幕淡入淡出。
- 避免无意义的闪白、旋转、抖动、过强锐化、重滤镜和满屏花字。
- 同一条视频建立有限的视觉语法：一套字幕、一到两种转场、一套色彩和音效风格。

## 7. 内容类型策略

- 口播：人物是主线，B-roll用于证据和示例；每次插入后及时回到人物建立信任。
- 短剧：先保表演和反应，再保对白；用反应镜头、J/L Cut和环境声维持空间关系。
- 产品/酒店方案：先展示结果或问题，再给场景证据、操作过程和可验证细节。
- 教程：录屏重点位置使用平滑推近和指示，不遮挡鼠标、按钮和关键数据。

## 8. 成片复核

1. 静音看前三秒。
2. 只听声音，检查叙事是否连续。
3. 倍速看，找重复、拖沓和无效镜头。
4. 原速逐帧检查切点、字幕、人物脸手和黑帧。
5. 用手机尺寸检查竖屏安全区和小字可读性。
